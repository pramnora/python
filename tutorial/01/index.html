Python Tutorial

Introduction

This is a Python programming tutorial.

The Python programming language can be downloaded; and, then, run locally from on your own computer at home.

In order to do that, you just need to visit: https://www.python.org/downloads/.

However, as a complete 'beginner'...; then, it's really not necessary to download anything, at least, to start out.

Instead, we are going to try and get used to learning to use the basic language, first; by typing in and running Python code inside of the web browser itself.

Go to: https://www.python.org.

On that first page you will see a yellow button: >_; click there...

The next screen should load up the 'Python Interactive prompt' that looks like so: >>>.

Typing in.../and, then, running Python code...is a 2 step process.

  1. You type Python code after the prompt: >>>
  2. and, then, run that code, quite simply, by pressing your keyboard's [ENTER] key

After the prompt type in some Python code as follows...

>>>1+1 (..next, press your keyboard's [ENTER] key; and, you should see the output result appear as...)
>>>2
>>>

NOTE: The bottom most prompt >>> just means the computer is still waiting for you to type in and run some more Python code.

Warmest congratulations, you've just managed to type in and run your first Python code very successfully.
The chinese say that, 'even the longest journey begins with the first step...!' ;-)
